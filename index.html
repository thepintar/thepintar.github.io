Duckies!

Duckies is a game based off of the popular "2048" and was a solo challenge I completed at DevBootcamp. It was one of the first javascript projects I undertook with emphasis on Object-oriented programming. As an extended challenge to myself I opted out of storing the values in a nested array, and instead stored them in one long array. The final code could use a refactor, but I believe it illustrates solid OOP.

Controls:

Up Arrow to shift all images up

Left Arrow to shift all images left

Down Arrow to shift all images down

Right Arrow to shift all images right
